# ドメイン操作ユースケース一覧

Bluegrass GONG のメインドメインである **ルーム** と **レパートリー** について、
UIからの操作・表示ユースケースを整理する。

---

## 1. ルーム (Room)

### 1.1 操作系

| ID | ユースケース | DB操作 | 実装状況 |
|:---|:-----------|:-------|:---------|
| R-1 | ルームを作成する | INSERT rooms + room_members | 実装済 |
| R-2 | ルームに参加する（コード入力） | SELECT rooms → INSERT room_members | 実装済 |
| R-3 | ルームに参加する（QR読み取り） | html5-qrcode でカメラスキャン → コード自動入力 | 実装済 |
| R-4 | マッチレベルを切り替える | クライアント側フィルタ（strict / normal / loose） | 実装済 |
| R-5 | ルームコードをコピーする | クリップボード操作 | 実装済 |
| R-6 | ルームコードをQR共有する | QR生成（qrcode.react） | 実装済 |
| R-7 | ルームから退出する | DELETE room_members | 実装済 |
| R-8 | ルームを削除する | DELETE rooms | **未実装**（#3でイシュー化済） |
| R-9 | 曲を演奏済みにマークする | INSERT room_played_songs（Realtime同期） | 実装済 |
| R-10 | 演奏済みマークを取り消す | DELETE room_played_songs（Realtime同期） | 実装済 |

### 1.2 表示系

| ID | ユースケース | 表示内容 | 実装状況 |
|:---|:-----------|:--------|:---------|
| R-D1 | ルーム情報を表示する | ルームコード（6文字）、参加者数 | 実装済 |
| R-D2 | メンバー一覧を表示する | 名前、アバター（Realtime更新） | 実装済（ボトムシート） |
| R-D3 | 共通曲リストを表示する | 曲タイトル、キー、カバー率、演奏済み状態 | 実装済 |
| R-D4 | 共通曲の詳細を表示する | メンバー別：楽器、サブパート、習熟度、ボーカル、得意キー | 実装済（カード展開） |
| R-D5 | 楽器構成サマリを表示する | PartIconラベル × 人数（Gt×2 Bj×1） | 実装済 |
| R-D6 | 習熟度インジケータを表示する | 色付きドット（緑=ready / 黄=practice / 灰=learning） | 実装済 |
| R-D7 | ボーカル構成を表示する | Lead / Har(H) / Har(L) / Inst | 実装済 |
| R-D8 | ルーム作成完了を表示する | コード、コピーボタン、QRコード、入室ボタン | 実装済 |
| R-D9 | ルームの有効期限を表示する | 残り時間（ヘッダーに表示、1分毎更新） | 実装済 |
| R-D10 | 参加中ルーム一覧を表示する | ホームに有効期限内のルーム一覧 | 実装済 |
| R-D11 | 演奏済みカウントを表示する | 「X曲 演奏済み」、済み曲は半透明+取り消し線+リスト下部 | 実装済 |

## 2. レパートリー (Repertoire)

### 2.1 操作系

| ID | ユースケース | DB操作 | 実装状況 |
|:---|:-----------|:-------|:---------|
| P-1 | 曲を検索する | SELECT songs（タイトル / ボーカル / キー / テンポ） | 実装済 |
| P-2 | 曲をレパートリーに追加する | INSERT user_repertoires（part, sub_parts, vocal, preferred_keys, proficiency） | 実装済 |
| P-3 | レパートリーを編集する | UPDATE user_repertoires（登録済み曲タップで編集シート表示） | 実装済 |
| P-4 | レパートリーから曲を削除する | DELETE user_repertoires（ゴミ箱アイコン） | 実装済 |
| P-5 | 習熟度をインライン変更する | UPDATE user_repertoires.proficiency（◎○△ワンタップ） | 実装済 |
| P-6 | お気に入りをトグルする | UPDATE user_repertoires.is_favorite（★ワンタップ） | 実装済 |
| P-7 | マイレパートリーをフィルタする | クライアント側（習熟度 / パート / ★のみ） | 実装済 |
| P-8 | マイレパートリーをソートする | クライアント側（★順 / A-Z / 習熟度 / 新着） | 実装済 |

### 2.2 表示系

| ID | ユースケース | 表示内容 | 実装状況 |
|:---|:-----------|:--------|:---------|
| P-D1 | 曲リストを表示する | タイトル、キー、テンポ、メイン楽器 | 実装済 |
| P-D2 | 検索結果件数を表示する | 「X曲 見つかりました」 | 実装済 |
| P-D3 | 登録済みステータスを表示する | ✓登録済（緑）/ +追加（ボタン） | 実装済 |
| P-D4 | フィルタ状態を表示する | ボーカル種別・キー・テンポの選択状態 | 実装済 |
| P-D5 | レパートリー数を表示する | プロフィール：カウント / ホーム：「X曲登録済み」 | 実装済 |
| P-D6 | 登録シートを表示する | 曲名、キー、パート（ツータップ方式）、ボーカル＋得意キー（親子）、習熟度 | 実装済 |
| P-D7 | マイレパートリー詳細を表示する | PartIconラベル、サブパート、ボーカル、習熟度インライン切替 | 実装済 |
