# 画面設計

## 画面一覧（MVP）
1. ログイン画面（Google OAuth）
2. ホーム（レパートリー / ルーム参加 / ルーム作成）
3. マイレパートリー（曲検索・追加 / 一覧・編集・削除）
4. ルーム作成（QRコード・部屋コード発行）
5. ルーム参加（コード入力 / QR読み取り）
6. ルーム内画面（メンバー一覧 / 3段階マッチング / 曲詳細展開）
7. プロフィール（名前・メインパート設定）

## 画面遷移

```text
[ログイン] → [ホーム]
                ├→ [マイレパートリー] → [曲検索・追加]
                ├→ [ルーム作成] → [ルーム内画面]
                ├→ [ルーム参加] → [ルーム内画面]
                └→ [プロフィール]

[ルーム内画面]
  ├→ メンバー一覧
  ├→ マッチング結果（Strict / Normal / Loose トグル）
  └→ 曲詳細（タップで展開）
```

## ナビゲーション構成

- ボトムナビ3タブ: ホーム / レパートリー / プロフィール
- ルーム内はボトムナビが切り替わる（メンバー / 退出）→ 没入型UI
